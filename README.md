## 📚 Ödevler

### ✅ Hafta 1 — Ödeme Sistemi
**Klasör:** `week1-payment-system/`

Modern bir ödeme sistemi uygulaması. SOLID prensipleri, Annotation, Reflection, AOP, Spring Boot ve React kullanılarak geliştirilmiştir.

**Kullanılan Teknolojiler:**
- Java 21, Spring Boot
- AOP (Aspect Oriented Programming)
- Custom Annotation + Reflection
- PostgreSQL + JPA
- React (Frontend)

**Öğrenilen Konular:**
- SOLID Prensipleri
- Custom Annotation + Reflection
- AOP ile loglama
- Katmanlı mimari (Controller → Service → Repository → Entity)
- PostgreSQL + JPA entegrasyonu
- React ile frontend geliştirme
